Python Package
The dataset, model source codes and pretrained weights can be obtained via a python package. Instlal with 
pip install breizhcrops
and run the minimal working example 

from breizhcrops import BreizhCrops, models
x, y, field_id = BreizhCrops("frh04")[0]
model = models.pretrained("Transformer")
y_pred = model(x.unsqueeze(0))
to get started.
